# Danish Registry Data Reference Tools

Standalone reference scripts for Danish public APIs. NOT part of the ResearchClaw pipeline.
These scripts use requests+pandas and must be run outside the experiment sandbox.

## real_registry_data_denmark.py
Fetches data from DST NavneBarometer, Statistikbanken (FOD, HFUDD11, INDKP101).
Run: python tools/real_registry_data_denmark.py
